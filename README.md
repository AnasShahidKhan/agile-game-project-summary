# Agile Game Development (Boulder Dash Recreation)
# Diamond Rush

This repository is a professional summary for my second-year Software Engineering team project at Swansea University.

**Note:** The original source code repository is private to protect the academic integrity of the module and the shared work of my 7 teammates.

---

### ## Project Overview

* **Project:** A modern recreation of the 1984 classic "Boulder Dash," built in Java.
* **Team:** 8 students.
* **Methodology:** 10-week Agile sprint cycle.
* **Goal:** To manage a team-based software project from conception to a polished, bug-free delivery.

---

### ## My Role & Key Contributions

These are the same bullets from my CV. As **Project Coordinator & Testing Manager**, I was responsible for:

* Acting as Project Coordinator & Testing Manager for the Agile team, overseeing sprint planning, task delegation, and QA, delivering the project 2 weeks ahead of schedule.
* Writing substantial portions of the core game logic in **Java**, implementing map generation, object interaction, and collision handling using **Object-Oriented Design** principles.
* Translating conceptual game designs into technical features and leading formal bug-tracking processes to ensure robust performance.
* Fostering a professional team environment through clear communication and collaborative problem-solving to boost development efficiency.
